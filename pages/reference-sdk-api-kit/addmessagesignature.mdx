import { Tabs } from 'nextra/components'

# `addMessageSignature`

Adds a signature to an existing message.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    import { apiKit } from './setup.ts'

    const safeMessageHash = '0x...'
    
    const signature = '0x...'

    await apiKit.addMessageSignature(safeMessageHash, signature)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import SafeApiKit from '@safe-global/api-kit'

    // How to get an Api key => http://docs.safe.global/sdk/api-kit/guides/how-to-use-api-keys
    export const apiKit = new SafeApiKit({
      chainId: 1n, // Mainnet
      apiKey: 'YOU_API_KEY'
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Parameters

### `safeMessageHash`

- **Type:** `string`

The Safe message hash.

```typescript focus=2
await apiKit.addMessageSignature(
  '0x...',
  '0x...'
)
```

### `signature`

- **Type:** `string`

The signature.

```typescript focus=3
await apiKit.addMessageSignature(
  '0x...',
  '0x...'
)
```
