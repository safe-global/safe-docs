# API Kit Reference

The [API Kit](../sdk/api-kit.mdx) facilitates the interaction with the [Safe Transaction Service API](../core-api/transaction-service-overview.mdx).

## Install dependencies

To add the API Kit to your project, run:

{/* <!-- vale off --> */}

<CH.Section>
  <CH.Code style={{ boxShadow: 'none' }}>
    ```bash pnpm
    pnpm add @safe-global/api-kit
    ```

    ```bash npm
    npm install @safe-global/api-kit
    ```

    ```bash yarn
    yarn add @safe-global/api-kit
    ```
  </CH.Code>
</CH.Section>
