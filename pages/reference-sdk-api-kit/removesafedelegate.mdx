# `removeSafeDelegate`

Removes a delegate for a given Safe address.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    import { DeleteSafeDelegateProps } from '@safe-global/api-kit'
    import { apiKit } from './setup.ts'

    const config: DeleteSafeDelegateProps = {
      delegateAddress: '0x...',
      delegatorAddress: '0x...',
      signer
    }

    await apiKit.removeSafeDelegate(config)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import SafeApiKit from '@safe-global/api-kit'

    // How to get an Api key => http://docs.safe.global/sdk/api-kit/guides/how-to-use-api-keys
    export const apiKit = new SafeApiKit({
      chainId: 1n, // Mainnet
      apiKey: 'YOU_API_KEY'
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Parameters

### `config.delegateAddress`

- **Type:** `string`

The address of the delegate that will be removed.

```typescript focus=2
await apiKit.removeSafeDelegate({
  delegateAddress: '0x...',
  delegatorAddress: '0x...',
  signer
})
```

### `config.delegatorAddress`

- **Type:** `string`

The delegator address.

```typescript focus=3
await apiKit.removeSafeDelegate({
  delegateAddress: '0x...',
  delegatorAddress: '0x...',
  signer
})
```

### `config.signer`

- **Type:** `Signer`

The signer.

```typescript focus=4
await apiKit.removeSafeDelegate({
  delegateAddress: '0x...',
  delegatorAddress: '0x...',
  signer
})
```
