import { Tabs } from 'nextra/components'

# `decodeData`

Decodes the specified Safe transaction data.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    import { apiKit } from './setup.ts'

    const data = '0x...'

    const decodedData = await apiKit.decodeData(data)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import SafeApiKit from '@safe-global/api-kit'

    export const apiKit = new SafeApiKit({
      chainId: 1n, // Mainnet
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<any>`

The transaction data decoded.

## Parameters

### `data`

- **Type:** `string`

The Safe transaction data to decode.

```typescript focus=2
const decodedData = await apiKit.decodeData(
  '0x...'
)
```
