# `proposeTransaction`

Creates a new multi-signature transaction with its confirmations and stores it in the Safe Transaction Service.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    import { ProposeTransactionProps } from '@safe-global/api-kit'
    import { OperationType, SafeTransactionData } from '@safe-global/types-kit'
    import { apiKit } from './setup.ts'

    const safeTransactionData: SafeTransactionData = {
      operation: OperationType.Call,
      safeTxGas: '0x...',
      baseGas: '123',
      gasPrice: '123',
      gasToken: '0x...',
      refundReceiver: '0x...',
      nonce: '10'
    }

    const config: ProposeTransactionProps = {
      safeAddress: '0x...',
      safeTxHash: '0x...',
      safeTransactionData,
      senderAddress: '0x...',
      senderSignature: '0x...',
      origin: 'App name' // Optional
    }

    await apiKit.proposeTransaction(config)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import SafeApiKit from '@safe-global/api-kit'

    import { OperationType, SafeTransactionData } from '@safe-global/types-kit'

    // How to get an Api key => http://docs.safe.global/core-api/how-to-use-api-keys
    export const apiKit = new SafeApiKit({
      chainId: 1n, // Mainnet
      apiKey: 'YOU_API_KEY'
    })
    ```
  </Tabs.Tab>
</Tabs>

## Parameters

### `config.safeAddress`

- **Type:** `string`

The Safe address.

```typescript focus=2
await apiKit.proposeTransaction({
  safeAddress: '0x...',
  safeTxHash: '0x...',
  safeTransactionData,
  senderAddress: '0x...',
  senderSignature: '0x...'
})
```

### `config.safeTxHash`

- **Type:** `string`

The Safe transaction hash of the proposed transaction.

```typescript focus=3
await apiKit.proposeTransaction({
  safeAddress: '0x...',
  safeTxHash: '0x...',
  safeTransactionData,
  senderAddress: '0x...',
  senderSignature: '0x...'
})
```
