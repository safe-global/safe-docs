# `getSafeOperationsByAddress`

Returns the list of SafeOperations associated to a Safe account.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    import { GetSafeOperationListProps } from '@safe-global/api-kit'
    import { apiKit } from './setup.ts'

    const config: GetSafeOperationListProps = {
      safeAddress: '0x...',
      ordering: 'created', // Optional
      limit: '10', // Optional
      offset: '50' // Optional
    }

    const safeOperationsResponse = await apiKit.getSafeOperationsByAddress(config)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import SafeApiKit from '@safe-global/api-kit'

    export const apiKit = new SafeApiKit({
      chainId: 1n, // Mainnet
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<GetSafeOperationListResponse>`

The paginated list of operations with `results` containing the `SafeOperationResponse` list.

## Parameters

### `safeAddress`

- **Type:** `string`

The Safe address.

```typescript focus=2
const safeOperationsResponse = await apiKit.getSafeOperationsByAddress({
  safeAddress: '0x...'
})
```

### `ordering` (Optional)

- **Type:** `string`

The field used when ordering the results. Can be one of: `user_operation__nonce`, `created`, `-user_operation__nonce`, `-created`. Default: `-user_operation__nonce`.

```typescript focus=4
const safeOperationsResponse = await apiKit.getSafeOperationsByAddress(
  {
    safeAddress: '0x...',
    ordering: 'created'
  }
)
```

### `limit` (Optional)

- **Type:** `number`

The number of results to return per page.

```typescript focus=4
const safeOperationsResponse = await apiKit.getSafeOperationsByAddress(
  {
    safeAddress: '0x...',
    limit: 10
  }
)
```

### `offset` (Optional)

- **Type:** `number`

The initial index from which to return the results.

```typescript focus=4
const safeOperationsResponse = await apiKit.getSafeOperationsByAddress(
  {
    safeAddress: '0x...',
    offset: 50
  }
)
```
