# `constructor`

Returns an instance of the API Kit.

## Usage

{/* <!-- vale off --> */}

```typescript
import SafeApiKit from '@safe-global/api-kit'

const chainId = 1n

const txServiceUrl = 'https://...'

// How to get an Api key => http://docs.safe.global/core-api/how-to-use-api-keys
const apiKit = new SafeApiKit({
  chainId,
  txServiceUrl, // Optional
  apiKey: 'YOU_API_KEY' //  Mandatory if txServiceUrl is not specified
})
```

{/* <!-- vale on --> */}

## Returns

`SafeApiKit`

A new instance of the API Kit.

## Parameters

### `chainId`

- **Type:** `bigint`

The chain ID.

```typescript focus=2
const apiKit = new SafeApiKit({
  chainId: 1n,
  apiKey: 'YOU_API_KEY'
})
```

### `apiKey`

- **Type:** `string`

The API key. This parameter is mandatory when using default Safe provided services. It can be omitted if using a custom Tx Service. [Check how to get one](../core-api/how-to-use-api-keys).

```typescript focus=3
const apiKit = new SafeApiKit({
  chainId: 1n,
  apiKey: 'YOU_API_KEY'
})
```

### `txServiceUrl` (Optional)

- **Type:** `string`

A custom Safe Transaction Service URL that is not supported by default.

```typescript focus=3
const apiKit = new SafeApiKit({
  chainId: 1n,
  txServiceUrl: 'https://...'
})
```
