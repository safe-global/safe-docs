### `getPendingTransactions`

Returns all the pending transactions associated with the Safe connected to the [`SafeProvider`](../safeprovider.mdx).

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.tsx']}>
  <Tabs.Tab>
    ```typescript
    import { useSafe } from '@safe-global/safe-react-hooks'

    function PendingTransactions() {
      const { getPendingTransactions } = useSafe()
      const { data, error, isLoading } = getPendingTransactions()

      if (isLoading) return <p>Loading...</p>
      
      if (error) return <p>Error: {error.message}</p>

      return JSON.stringify(data, null, 2)
    }

    export default PendingTransactions
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`UsePendingTransactionsReturnType`

- A [query result](https://tanstack.com/query/v5/docs/framework/react/reference/useQuery) object containing the pending transactions in the `data` property.
- The `data` property type is `SafeMultisigTransaction[]` and contains the [transaction information](https://github.com/safe-global/safe-core-sdk/blob/dc602ed126cc210048b278b2372c0979ee4be62b/packages/types-kit/src/types.ts#L216C1-L247C2) extracted from the Transaction Service.

## Parameters

- **Type**: `UsePendingTransactionsParams`

Parameters to customize the hook behavior.

```typescript
const result = getPendingTransactions(params)
```

### `config` (Optional)

- **Type**: [`SafeConfig`](../safeconfig.mdx)

The configuration used instead of the one from the nearest [`SafeProvider`](../safeprovider.mdx).
