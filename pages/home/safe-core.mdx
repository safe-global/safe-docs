# Safe\{Core\}

Safe\{Core\} is an open-source and modular [Account Abstraction](../glossary.md#account-abstraction) stack that brings a flexible developer experience. It serves as the essential tooling and infrastructure for integrating the Safe Smart Account into any digital platform, ensuring a tested core with flexible and secure capabilities for innovative applications.

The Safe\{Core\} stack is categorized into three distinct groups:
- Safe\{Core\} Account Abstraction SDK
- Safe\{Core\} API
- Safe Smart Account
