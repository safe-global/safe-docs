# `revokeGuardianWithThreshold` (0x936f7d86)



Lets an authorised module revoke a guardian from a wallet and change the threshold.




Defined in [`GuardianStorage.sol`](https://github.com/safe-global/safe-modules/tree/recovery/v0.1.0/modules/recovery/contracts/generated-markups/candide-contracts/contracts/modules/social_recovery/storage/GuardianStorage.sol#Lundefined).
