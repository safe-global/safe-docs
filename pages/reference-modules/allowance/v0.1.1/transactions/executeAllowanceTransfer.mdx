# `executeAllowanceTransfer` (0x4515641a)



Allows to use the allowance to perform a transfer.




Defined in [`AllowanceModule.sol`](https://github.com/safe-global/safe-modules/tree/allowance/v0.1.1/modules/allowances/contracts/AllowanceModule.sol#L152).
