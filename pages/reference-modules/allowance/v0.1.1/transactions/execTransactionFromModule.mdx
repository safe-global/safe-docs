# `execTransactionFromModule` (0x468721a7)



Allows a Module to execute a Safe transaction without any further confirmations.




Defined in [` ISafe.sol`](https://github.com/safe-global/safe-modules/tree/allowance/v0.1.1/modules/allowances/contracts/ISafe.sol#Lundefined)
