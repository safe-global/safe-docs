# `removeDelegate` (0xdd43a79f)



Allows to remove a delegate.




Defined in [`AllowanceModule.sol`](https://github.com/safe-global/safe-modules/tree/allowance/v0.1.1/modules/allowances/contracts/AllowanceModule.sol#L324).

        function removeDelegate(address delegate, bool removeAllowances) public

## Parameters

### `delegate`

  - **Type:** `address`

  Delegate that should be removed.

  ```solidity focus=2
  (ISafe safe).removeDelegate(
    0x...,
    true
  );
  ```


### `removeAllowances`

  - **Type:** `bool`

  Indicator if allowances should also be removed. This should be set to `true` unless this causes an out of gas, in this case the allowances should be "manually" deleted via `deleteAllowance`.

  ```solidity focus=3
  (ISafe safe).removeDelegate(
    0x...,
    true
  );
  ```
