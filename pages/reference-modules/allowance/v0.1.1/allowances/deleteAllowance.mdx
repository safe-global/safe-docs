# `deleteAllowance` (0x885133e3)



Allows to remove the allowance for a specific delegate and token. This will set all values except the `nonce` to 0.




Defined in [` AllowanceModule.sol`](https://github.com/safe-global/safe-modules/tree/allowance/v0.1.1/modules/allowances/contracts/AllowanceModule.sol#L133)
