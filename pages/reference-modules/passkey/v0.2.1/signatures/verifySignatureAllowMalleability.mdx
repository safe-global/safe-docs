# `verifySignatureAllowMalleability` 



Verifies the signature of a message using P256 elliptic curve, without signature
malleability check.




Defined in [` P256.sol`](https://github.com/safe-global/safe-modules/tree/passkey/v0.2.1/modules/passkey/contracts/libraries/P256.sol#L100)


## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function verifySignatureAllowMalleability(IP256Verifier verifier,bytes32 message,uint256 r,uint256 s,uint256 x,uint256 y) internal view returns (bool success)
    }

        function example() ... {

                IP256Verifier,
    ```

## Returns

### `success`

  - **Type:** `bool`

  A boolean indicating whether the signature is valid or not.
