# Create a new Safe

You can create a new Safe by running the following command:

```bash
safe-creator <node_url> <private_key> --owners <checksummed_address_1> <checksummed_address_2> --threshold <uint> --salt-nonce <uint256>
```

## Example

```bash
safe-creator https://sepolia.infura.io/v3/$INFURA_TOKEN $PRIVATE_KEY --owners $OWNER_ADDRESS_1 [$OWNER_ADDRESS_2] --threshold 1
```

## Demo

The `PRIVATE_KEY` environment variable for this demo was set to an _EOA_ private key.
