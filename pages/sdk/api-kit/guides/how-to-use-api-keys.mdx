# How to use API Keys

## Getting a Safe API Key

To use Safe API services, you'll need an API key. Visit our [dashboard](https://developer.safe.global) to create an account and generate your API key.

## Restrict Access to Your API Keys

You can limit how your API keys are used by modifying their permissions on the [API Keys page](https://developer.safe.global/api-keys). Restrictions can include:

- IP addresses: Specify which IP addresses are allowed to make requests.
- User agents: Limit access to specific browsers, SDK versions, or other user agents.
- Origins: Define which domains are permitted to make requests.

## Using Your API Key

To use your API key, include it in the `Authorization` header of your requests. Here's several examples using different options:

        {
          method: 'GET',
          headers: {
            Authorization: 'Bearer YOUR_API_KEY'
          }
        }
      )
        .then(response => response.json())
        .then(data => console.log(data))
        .catch(error => console.error('Error:', error))
      ```

      const multisigTransactions = await apiKit.getMultisigTransactions(

      )

      ```
