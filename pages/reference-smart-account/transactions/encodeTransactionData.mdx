# `encodeTransactionData`

Encodes the transaction data for `execTransaction`.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function encodeTransactionData(
            address to,
            uint256 value,
            bytes data,
            Enum.Operation operation,
            uint256 safeTxGas,
            uint256 baseGas,
            uint256 gasPrice,
            address gasToken,
            address refundReceiver,
            uint256 _nonce
        ) external view returns (bytes);
    }

    contract Example {
        function example() ... {
            (ISafe safe).encodeTransactionData(
                0x...,
                0,
                "0x...",
                Enum.Operation.Call,
                100000,
                0,
                0,
                0x...,
                0x...,
                1
            );
        }
    }
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

- **Type:** `bytes`

Encoded transaction data.
