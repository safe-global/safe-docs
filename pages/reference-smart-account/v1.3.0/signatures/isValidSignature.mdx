# `isValidSignature` (0x1626ba7e)



Implementation of updated EIP-1271

Should return whether the signature provided is valid for the provided data.
The save does not implement the interface since `checkSignatures` is not a view method.
The method will not perform any state changes (see parameters of `checkSignatures`)




Defined in [`CompatibilityFallbackHandler.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/handler/CompatibilityFallbackHandler.sol#L28).

## Returns



  - **Type:** `bytes4`

  a bool upon valid or invalid signature with corresponding _dataHash




## Parameters

### `_dataHash`

  - **Type:** `bytes32`

  Hash of the data signed on the behalf of address(msg.sender)

  ```solidity focus=2
  ISafe(0x...).isValidSignature(
    "0x...",
    "0x..."
  );
  ```


### `_signature`

  - **Type:** `bytes`

  Signature byte array associated with _dataHash

  ```solidity focus=3
  ISafe(0x...).isValidSignature(
    "0x...",
    "0x..."
  );
  ```
