# `checkSignatures` (0x934f3a11)



Checks whether the signature provided is valid for the provided data, hash. Will revert otherwise.




Defined in [`GnosisSafe.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/GnosisSafe.sol#L221).


## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function checkSignatures(
            bytes32 dataHash,
            bytes memory data,
            bytes memory signatures
        ) public view
    }

    contract Example {
        function example() ... {
            (ISafe safe).checkSignatures(
                "0x...",
                "0x...",
                "0x..."
            );
        }
    }
    ```

## Parameters

### `dataHash`

  - **Type:** `bytes32`

  Hash of the data (could be either a message hash or transaction hash)

  ```solidity focus=2
  (ISafe safe).checkSignatures(
    "0x...",
    "0x...",
    "0x..."
  );
  ```


### `data`

  - **Type:** `bytes`

  That should be signed (this is passed to an external validator contract)

  ```solidity focus=3
  (ISafe safe).checkSignatures(
    "0x...",
    "0x...",
    "0x..."
  );
  ```


### `signatures`

  - **Type:** `bytes`

  Signature data that should be verified. Can be ECDSA signature, contract signature (EIP-1271) or approved hash.

  ```solidity focus=4
  (ISafe safe).checkSignatures(
    "0x...",
    "0x...",
    "0x..."
  );
  ```
