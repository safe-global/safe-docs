# `checkNSignatures` (0x12fb68e0)



Checks whether the signature provided is valid for the provided data, hash. Will revert otherwise.




Defined in [` GnosisSafe.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/GnosisSafe.sol#L240)

## Parameters

### `dataHash`

  - **Type:** `bytes32`

  Hash of the data (could be either a message hash or transaction hash)

  ```solidity focus=2
  (ISafe safe).checkNSignatures(
    "0x...",
    0
  );
  ```


### `requiredSignatures`

  - **Type:** `uint256`

  Amount of required valid signatures.

  ```solidity focus=3
  (ISafe safe).checkNSignatures(
    "0x...",
    0
  );
  ```
