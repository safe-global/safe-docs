# `setGuard` (0xe19a9dd9)



Set a guard that checks transactions before execution




Defined in [`GuardManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/base/GuardManager.sol#L34).


## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function setGuard(address guard) external authorized
    }

    contract Example {
        function example() ... {
            ISafe(0x...).setGuard(
                0x...
            );
        }
    }
    ```
  </Tabs.Tab>

## Parameters

### `guard`

  - **Type:** `address`

  The address of the guard to be used or the 0 address to disable the guard

  ```solidity focus=2
  ISafe(0x...).setGuard(
    0x...
  );
  ```
