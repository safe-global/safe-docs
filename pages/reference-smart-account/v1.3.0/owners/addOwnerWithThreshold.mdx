# `addOwnerWithThreshold` (0x0d582f13)



Adds the owner `owner` to the Safe and updates the threshold to `_threshold`.

Allows to add a new owner to the Safe and update the threshold at the same time.
<OnlySafeTxCallout />




Defined in [` OwnerManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/base/OwnerManager.sol#L51)
