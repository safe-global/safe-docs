# `setFallbackHandler` (0xf08a0323)



Allows to add a contract to handle fallback calls.
Only fallback calls without value and with data will be forwarded.
<OnlySafeTxCallout />




Defined in [` FallbackManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/base/FallbackManager.sol#L26)


## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>

    interface ISafe {
        function setFallbackHandler(address handler) public authorized
