# `execTransactionFromModule` (0x468721a7)



Allows a Module to execute a Safe transaction without any further confirmations.




Defined in [` ModuleManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/base/ModuleManager.sol#L61)
