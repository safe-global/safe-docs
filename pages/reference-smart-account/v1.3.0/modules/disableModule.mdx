# `disableModule` (0xe009cfde)



Disables the module `module` for the Safe.

Allows to remove a module from the whitelist.
<OnlySafeTxCallout />




Defined in [` ModuleManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/base/ModuleManager.sol#L47)
