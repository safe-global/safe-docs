# `simulate` (0xbd61951d)



Performs a delegetecall on a targetContract in the context of self.
Internally reverts execution to avoid side effects (making it static). Catches revert and returns encoded result as bytes.




Defined in [` CompatibilityFallbackHandler.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/handler/CompatibilityFallbackHandler.sol#L87)
