# `requiredTxGas` (0xc4ca3a9c)



Deprecated in favor of common/StorageAccessible.sol and will be removed in next version.
Allows to estimate a Safe transaction.
This method is only meant for estimation purpose, therefore the call will always revert and encode the result in the revert data.
Since the `estimateGas` function includes refunds, call this method to get an estimated of the costs that are deducted from the safe with `execTransaction`




Defined in [` GnosisSafe.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/GnosisSafe.sol#L315)

## Returns



  - **Type:** `uint256`

  Estimate without refunds and overhead fees (base transaction and payload data gas costs).




## Parameters

### `to`

  - **Type:** `address`

  Destination address of Safe transaction.

  ```solidity focus=2
  (ISafe safe).requiredTxGas(
    0x...,
    0
  );
  ```


### `operation`

  - **Type:** `enum Enum.Operation`

  Operation type of Safe transaction.

    0x...,
