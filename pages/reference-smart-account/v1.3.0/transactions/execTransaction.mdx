# `execTransaction` (0x6a761202)



Allows to execute a Safe transaction confirmed by required number of owners and then pays the account that submitted the transaction.
Note: The fees are always transferred, even if the user transaction fails.




Defined in [` GnosisSafe.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.3.0/contracts/GnosisSafe.sol#L111)
