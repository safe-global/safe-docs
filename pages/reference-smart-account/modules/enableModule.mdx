# `enableModule`

Enables a new Safe Module in a Safe account.

<Callout type='warning'>This action can only be done via a Safe transaction.</Callout>

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function enableModule(address module) external;
    }

    contract Example {
        function example() ... {
            (ISafe safe).enableModule("0x...");
        }
    }
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Parameters

### `module`

- **Type:** `address`

Safe Module to be enabled.

```solidity focus=2
(ISafe safe).enableModule(
    "0x..."
);
```
