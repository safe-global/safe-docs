# `checkNSignatures` (0x12fb68e0)



Checks whether the signature provided is valid for the provided data and hash. Reverts otherwise.

<ReentrancyCallout />




Defined in [`Safe.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/Safe.sol#L274).


## Usage

    interface ISafe {
        function checkNSignatures(
            bytes32 dataHash,
            bytes memory data,

## Parameters

### `dataHash`

  - **Type:** `bytes32`

  Hash of the data (could be either a message hash or transaction hash)

  ```solidity focus=2
  (ISafe safe).checkNSignatures(
    "0x...",
    "0x...",
    "0x...",
    0
  );
  ```


### `data`

  - **Type:** `bytes`

  That should be signed (this is passed to an external validator contract)

  ```solidity focus=3
  (ISafe safe).checkNSignatures(
    "0x...",
    "0x...",
    "0x...",
    0
  );
  ```


### `signatures`

  - **Type:** `bytes`

  Signature data that should be verified. Can be packed ECDSA signature (\{bytes32 r\}\{bytes32 s\}\{uint8 v\}), contract signature (EIP-1271) or approved hash.

  ```solidity focus=4
  (ISafe safe).checkNSignatures(
    "0x...",
    "0x...",
    "0x...",
    0
  );
  ```


### `requiredSignatures`

  - **Type:** `uint256`

  Amount of required valid signatures.

  ```solidity focus=5
  (ISafe safe).checkNSignatures(
    "0x...",
    "0x...",
    "0x...",
    0
  );
  ```
