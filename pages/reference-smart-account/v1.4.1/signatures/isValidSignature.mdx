## Returns



  - **Type:** `bytes4`

  The EIP-1271 magic value.




## Parameters

### `_data`

  - **Type:** `bytes`

  Arbitrary length data signed on the behalf of address(msg.sender).

  ```solidity focus=2
  (ISafe safe).isValidSignature(
    "0x...",
    "0x..."
  );
  ```


### `_signature`

  - **Type:** `bytes`

  Signature byte array associated with _data.

  ```solidity focus=3
  (ISafe safe).isValidSignature(
    "0x...",
    "0x..."
  );
  ```
