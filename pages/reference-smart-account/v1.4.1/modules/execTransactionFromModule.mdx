# `execTransactionFromModule` (0x468721a7)

Executes an `operation` (`0`: Call, `1`: DelegateCall) to `to` address with the value `value` (in native token)

Defined in [`ModuleManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/base/ModuleManager.sol#L81).

### `success`

  - **Type:** `bool`

  Boolean flag indicating if the call succeeded.
