## Parameters

### `prevModule`

  - **Type:** `address`

  Previous module in the modules linked list.

  ```solidity focus=2
  (ISafe safe).disableModule(
    0x...,
    0x...
  );
  ```


### `module`

  - **Type:** `address`

  Module to be removed.

  ```solidity focus=3
  (ISafe safe).disableModule(
    0x...,
    0x...
  );
  ```
