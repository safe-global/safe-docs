# `getModulesPaginated` (0xcc2f8452)



Returns an array of modules.
If all entries fit into a single page, the next pointer will be 0x1.
If another page is present, next will be the last element of the returned array.




Defined in [`ModuleManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/base/ModuleManager.sol#L144).

## Returns

### `array`

  - **Type:** `address[]`

  Array of modules.


### `next`

  - **Type:** `address`

  Start of the next page.




## Parameters

### `start`

  - **Type:** `address`

  Start of the page. Has to be a module or start pointer (0x1 address)

  ```solidity focus=2
  (ISafe safe).getModulesPaginated(
    0x...,
    0
  );
  ```


### `pageSize`

  - **Type:** `uint256`

  Maximum number of modules that should be returned. Has to be > 0

  ```solidity focus=3
  (ISafe safe).getModulesPaginated(
    0x...,
    0
  );
  ```
