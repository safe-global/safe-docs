# `createProxyWithCallback` (0xd18af54d)



Deploy a new proxy with `_singleton` singleton and `saltNonce` salt.
Optionally executes an initializer call to a new proxy and calls a specified callback address `callback`.




Defined in [` SafeProxyFactory.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/proxies/SafeProxyFactory.sol#L86)

## Parameters

### `_singleton`

  - **Type:** `address`

  Address of singleton contract. Must be deployed at the time of execution.

  ```solidity focus=2
  (ISafe safe).createProxyWithCallback(
    0x...,
    
  );
  ```


### `callback`

  - **Type:** `contract IProxyCreationCallback`

  Callback that will be invoked after the new proxy contract has been successfully deployed and initialized.

    0x...,
