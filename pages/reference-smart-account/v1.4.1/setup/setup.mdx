# `setup` (0xb63e800d)

Sets an initial storage of the Safe contract.

<Callout type='warning'>

This method can only be called once. If a proxy was created without setting up, anyone can call setup and claim the proxy.
</Callout>

Defined in [`Safe.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/Safe.sol#L95).

## Parameters

### `_owners`

  - **Type:** `address[]`

  List of Safe owners.

  ```solidity focus=2

    [0x...],
    0,
    0x...,
    "0x...",
    0x...,
    0x...,
    0,
    0x...
  );
  ```


### `_threshold`

  - **Type:** `uint256`

  Number of required confirmations for a Safe transaction.
