# `createProxyWithNonce` (0x1688f0b9)



Deploys a new proxy with `_singleton` singleton and `saltNonce` salt. Optionally executes an initializer call to a new proxy.




Defined in [` SafeProxyFactory.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/proxies/SafeProxyFactory.sol#L52)

        function createProxyWithNonce(address _singleton,bytes memory initializer,uint256 saltNonce) public returns (SafeProxy proxy)

## Parameters

### `_singleton`

  - **Type:** `address`

  Address of singleton contract. Must be deployed at the time of execution.

  ```solidity focus=2
  (ISafe safe).createProxyWithNonce(
    0x...,
    "0x...",
    0
  );
  ```


### `initializer`

  - **Type:** `bytes`

  Payload for a message call to be sent to a new proxy contract.

  ```solidity focus=3
  (ISafe safe).createProxyWithNonce(
    0x...,
    "0x...",
    0
  );
  ```


### `saltNonce`

  - **Type:** `uint256`

  Nonce that will be used to generate the salt to calculate the address of the new proxy contract.

  ```solidity focus=4
  (ISafe safe).createProxyWithNonce(
    0x...,
    "0x...",
    0
  );
  ```
