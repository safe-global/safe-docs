# `setGuard` (0xe19a9dd9)



Set Transaction Guard `guard` for the Safe. Make sure you trust the guard.

Set a guard that checks transactions before execution
<OnlySafeTxCallout />
⚠️ IMPORTANT: Since a guard has full power to block Safe transaction execution,
a broken guard can cause a denial of service for the Safe. Make sure to carefully
audit the guard code and design recovery mechanisms.




Defined in [` GuardManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/base/GuardManager.sol#L53)

## Parameters

### `guard`

  - **Type:** `address`

  The address of the guard to be used or the 0 address to disable the guard

  ```solidity focus=2
  (ISafe safe).setGuard(
    0x...
  );
  ```
