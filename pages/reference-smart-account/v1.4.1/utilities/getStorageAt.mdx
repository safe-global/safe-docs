# `getStorageAt` (0x5624b25b)



Reads `length` bytes of storage in the currents contract




Defined in [`StorageAccessible.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/common/StorageAccessible.sol#L17).


## Usage

  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function getStorageAt(
            uint256 offset,
            uint256 length
        ) public view returns (bytes memory)
    }
    ```

## Returns



  - **Type:** `bytes`

  the bytes that were read.




## Parameters

### `offset`

  - **Type:** `uint256`

  - the offset in the current contract's storage in words to start reading from

  ```solidity focus=2
  (ISafe safe).getStorageAt(
    0,
    0
  );
  ```


### `length`

  - **Type:** `uint256`

  - the number of words (32 bytes) of data to read

  ```solidity focus=3
  (ISafe safe).getStorageAt(
    0,
    0
  );
  ```
