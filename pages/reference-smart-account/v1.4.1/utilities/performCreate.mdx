# `performCreate` (0x4c8c9ea1)



Deploys a new contract using the create opcode.




Defined in [`CreateCall.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/libraries/CreateCall.sol#L21).


## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function performCreate(
            uint256 value,
            bytes memory deploymentData
        ) public returns (address newContract)
    }

    contract Example {
        function example() ... {
            (ISafe safe).performCreate(
                0,
                "0x..."
            );
        }
    }
    ```
  </Tabs.Tab>
</Tabs>

## Returns

### `newContract`

  - **Type:** `address`

  The address of the newly created contract.




## Parameters

### `value`

  - **Type:** `uint256`

  The value in wei to be sent with the contract creation.

  ```solidity focus=2
  (ISafe safe).performCreate(
    0,
    "0x..."
  );
  ```


### `deploymentData`

  - **Type:** `bytes`

  The initialisation code of the contract to be created.

  ```solidity focus=3
  (ISafe safe).performCreate(
    0,
    "0x..."
  );
  ```
