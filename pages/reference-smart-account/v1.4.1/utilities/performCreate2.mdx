# `performCreate2` (0x4847be6f)



Deploys a new contract using the create2 opcode.




Defined in [` CreateCall.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/libraries/CreateCall.sol#L21)

## Returns

### `newContract`

  - **Type:** `address`

  The address of the newly created contract.




## Parameters

### `value`

  - **Type:** `uint256`

  The value in wei to be sent with the contract creation.

  ```solidity focus=2
  (ISafe safe).performCreate2(
    0
  );
  ```
