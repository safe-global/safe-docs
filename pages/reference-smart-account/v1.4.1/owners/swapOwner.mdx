# `swapOwner` (0xe318b52b)



Replaces the owner `oldOwner` in the Safe with `newOwner`.

<OnlySafeTxCallout />




Defined in [`OwnerManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/base/OwnerManager.sol#L99).

## Parameters

### `prevOwner`

  - **Type:** `address`

  Owner that pointed to the owner to be replaced in the linked list

  ```solidity focus=2

    0x...,
    0x...
  );
  ```

  ISafe(0x...).swapOwner(

    0x...,
    0x...
  );
