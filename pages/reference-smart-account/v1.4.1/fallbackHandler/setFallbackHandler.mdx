# `setFallbackHandler` (0xf08a0323)



Set Fallback Handler to `handler` for the Safe.

Only fallback calls without value and with data will be forwarded.
<OnlySafeTxCallout />
Cannot be set to the Safe itself.




Defined in [`FallbackManager.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/base/FallbackManager.sol#L50).


## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        function setFallbackHandler(address handler) public authorized
    }

    contract Example {
        function example() ... {
    ```

### `handler`

  - **Type:** `address`
