# `simulate` (0x1c5fb211)

Simulates a Safe transaction and returns the used gas, success boolean and the return data. This has to be called via a `delegatecall`.

It returns the data as `abi.encode(uint256(etimate), bool(success), bytes(returnData))`.

Defined in [`SimulateTxAccessor.sol`](https://github.com/safe-global/safe-smart-account/tree/v1.4.1/contracts/accessors/SimulateTxAccessor.sol#L42).

## Returns

### `estimate`

  - **Type:** `uint256`

  Gas used.


### `success`

  - **Type:** `bool`

  Success boolean value.


### `returnData`

  - **Type:** `bytes`

  Return data.
