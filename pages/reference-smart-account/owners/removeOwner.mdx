# `removeOwner`

## Overview

Removes the owner `owner` from the Safe and updates the threshold to `_threshold`.

<Callout type='warning'>This action can only be done via a Safe transaction.</Callout>

## Parameters

### `prevOwner`

- **Type:** `address`

Current owner address.

```solidity focus=2
(ISafe safe).removeOwner(
    "0x...",
    "0x...",
    1
);
```

### `owner`

- **Type:** `address`

Owner address to be removed.

```solidity focus=3
(ISafe safe).removeOwner(
    "0x...",
    "0x...",
    1
);
```

### `_threshold`

- **Type:** `uint256`

New threshold.

```solidity focus=4
(ISafe safe).removeOwner(
    "0x...",
    "0x...",
    1

```
