# `receive`

Receive function accepts native token transactions.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.sol']}>
  <Tabs.Tab>
    ```solidity
    interface ISafe {
        receive() external payable;
    }

    contract Example {
        function example() ... {
            (ISafe safe).receive();
        }
    }
    ```
  </Tabs.Tab>
</Tabs>

## Events

### `SafeReceived`

```solidity
event SafeReceived(address sender, uint256 value)
```

Emitted when the Safe contract receives a payment.
