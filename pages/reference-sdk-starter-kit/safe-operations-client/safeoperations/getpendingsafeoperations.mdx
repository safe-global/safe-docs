# `getPendingSafeOperations`

Retrieves the pending Safe operations for the current Safe account

## Usage

{/* <!-- vale off --> */}

<Tabs items={["getPendingTransactions.ts", "setup.ts"]}>
  <Tabs.Tab>
  ```typescript 
    const pendingSafeOperations = await safeClientWithSafeOperation.getPendingSafeOperations()
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```typescript 
      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })

      const safeOperationsClient = await safeClient.extend(safeOperations({
          bundlerUrl
        },
        {
          isSponsored,
          paymasterUrl
        }
      }))
  ```
  </Tabs.Tab>
</Tabs>

## Returns

`GetSafeOperationListResponse`

A paginated list of the pending SafeOperations for the current Safe account.
