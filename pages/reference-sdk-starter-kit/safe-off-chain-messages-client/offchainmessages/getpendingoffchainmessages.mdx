# `getPendingOffChainMessages`

This method returns the list of pending off-chain messages that need to be confirmed using the `confirmOffChainMessage()` method.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["get-pending-off-chain-messages.ts", "setup.ts"]}>
  <Tabs.Tab>
  ```typescript 
    const pendingMessages = await safeMessageClient.getPendingOffChainMessages()
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```typescript 
    const safeClient = await createSafeClient({
      provider,  
      signer,
      safeAddress
    })

    const safeMessageClient = safeClient.extend(offChainMessages())
  ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`SafeMessageListResponse`

A paginated list of the pending off-chain messages
