# `offChainMessages`

The `offChainMesssages` is an extension method that enables the SafeClient to work with off-chain messages.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["off-chain-messages.ts", "setup.ts"]}>
  <Tabs.Tab>
  ```typescript 
    import { offChainMessages } from '@safe-global/sdk-starter-kit'

    const safeClientWithMessages = safeClient.extend(offChainMessages())

    const txResult = await safeClientWithMessages.sendOffChainMessage({ message })
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```typescript 
    const safeClient = await createSafeClient({
      provider,
      signer,
      safeAddress
    })
  ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<SafeClient & { sendOffChainMessage, confirmOffChainMessage, getPendingOffChainMessages }>`

It returns the SafeClient with some additional methods enabling the off-chain messages usage.
