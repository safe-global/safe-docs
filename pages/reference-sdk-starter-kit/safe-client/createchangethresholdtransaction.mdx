# `createChangeThresholdTransaction`

Creates a transaction (to, value, data) to change the Safe account threshold. The transaction can be included individually or in a batch within the `send` method.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['create-change-threshold-transaction.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    const transaction = await safeClient.createChangeThresholdTransaction({
      threshold
    })

    const txResult = await safeClient.send({ transactions: [transaction] })
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
      import { createSafeClient } from '@safe-global/sdk-starter-kit'
          
      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<TransactionBase>`

Returns a `TransactionBase` transaction that you can use or include in a batch using the `send` method.

## Parameters

### `threshold`

- **Type:** `string`

The new threshold for the Safe account.
