# `getOwnersWhoApprovedTx`

Returns a list of owners who have approved a specific Safe transaction.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    import { safeClient } from './setup.ts'

    const safeTxHash = '0x...'

    const owners = safeClient.getOwnersWhoApprovedTx(safeTxHash)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import { createSafeClient } from '@safe-global/sdk-starter-kit'

    export const safeClient = await createSafeClient({
      provider,
      signer,
      safeAddress
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<string[]>`

The list of owners who approved the concrete `safeTxHash`.

## Parameters

### `safeTxHash`

- **Type:** `string`

The Safe transaction hash to check.
