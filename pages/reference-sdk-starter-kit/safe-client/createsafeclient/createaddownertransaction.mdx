# `createAddOwnerTransaction`

Creates a transaction (to, value, data) to add an owner to the Safe.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['one.ts', 'two.ts']}>
  <Tabs.Tab>
    ```typescript
    const transaction = await safeClient.createAddOwnerTransaction({
      ownerAddress,
      threshold
    })

    const txResult = await safeClient.send({ transactions: [transaction] })
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<TransactionBase>`

Returns a `TransactionBase` transaction that can be used/included in a batch with the `send` method

## Parameters

### `ownerAddress`

- **Type:** `string`

The ethereum address to add as an owner.

### `threshold` (optional)

- **Type:** `string`

The new threshold for the Safe account.
