# `extend`

The `extend` function plays a central role in the extensibility of the `SafeClient`. 

You can add any functionality you need to the `SafeClient` on your own or by using some of the SDK-implemented extensions (`onChainMessages`, `offChainMessages`, `safeOperations`).

## Usage

## Returns

`Promise<SafeClient & T> | SafeClient & T`

The SafeClient now includes the new functionality. It supports both asynchronous and synchronous usage.

## Parameters

### `(client: SafeClient) => Promise<T> | (client: SafeClient) => T`

- **Type:** `function`

The extension function will receive the current SafeClient and merge it with the new returned functionality.
