# `extend`

The extend function plays a central role in the extensibility of the SafeClient. 

You can add any functionality you need to the SafeClient on your own or by using some of the SDK implemented extensions.

## Usage

## Returns

`Promise<SafeClient & T> | SafeClient & T`

It returns the SafeClient with the new `T` functionality. IT can be used as a Promise as well

## Parameters

### `(client: SafeClient) => Promise<T> | (client: SafeClient) => T`

- **Type:** `function`

The extension function will receive the current SafeClient and will merge it with the new returned functionality.
