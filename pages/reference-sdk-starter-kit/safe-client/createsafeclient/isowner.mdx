# `isOwner`

This method checks if the provided address is an owner of the Safe.

## Usage

      const safeClient = await createSafeClient({

  </Tabs.Tab>

## Parameters

### `ownerAddress`

- **Type:** `string`

The ethereum address to check if it is an owner of the Safe.

## Returns

`Promise<boolean>`

A boolean value indicating if the provided address is an owner of the Safe.
