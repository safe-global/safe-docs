# `getAddress`

This method retrieves the current Safe address.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["send.ts", "setup.ts"]}>
  <Tabs.Tab>
    ```typescript 
      const nonce = await safeClient.getAddress()
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript 
      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<string>`

The address. It can be the predicted address in case the safe is not deployed yet and you use the `safeOptions` to instantiate the SafeClient.
