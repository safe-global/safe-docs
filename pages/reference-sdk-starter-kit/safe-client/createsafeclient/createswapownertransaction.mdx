# `createSwapOwnerTransaction`

Creates a transaction (to, value, data) to swap an owner of the Safe account. The transaction can be included individually or in a batch within the `send` method.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['create-swap-owner-transaction.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    const transaction = await safeClient.createSwapOwnerTransaction({
      oldOwnerAddress,
      newOwnerAddress
    })

    const txResult = await safeClient.send({ transactions: [transaction] })
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
      import { createSafeClient } from '@safe-global/sdk-starter-kit'
          
      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<TransactionBase>`

Returns a `TransactionBase` transaction that you can use or include in a batch using the `send` method.

## Parameters

### `oldOwnerAddress`

- **Type:** `string`

The Ethereum address to remove as an owner.

### `newOwnerAddress` (optional)

- **Type:** `string`

The new Ethereum address to add as an owner.
