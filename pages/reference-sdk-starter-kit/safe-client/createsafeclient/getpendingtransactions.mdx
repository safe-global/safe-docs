# `getPendingTransactions`

This method returns the list of pending transactions that need to be confirmed using the `confirm()` method to be executed in the blockchain.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["getPendingTransactions.ts", "setup.ts"]}>

  ```typescript 
    const pendingTransactions = await safeClient.getPendingTransactions()
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```typescript 
    const safeClient = await createSafeClient({

      safeAddress
    })
  ```
  </Tabs.Tab>
</Tabs>

## Returns

`SafeMultisigConfirmationListResponse`

A paginated list of the pending transactions
