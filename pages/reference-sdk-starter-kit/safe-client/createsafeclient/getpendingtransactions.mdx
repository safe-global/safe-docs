# `getPendingTransactions`

This method returns the list of pending transactions that require confirmation using the `confirm()` method to execute them on the blockchain.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["get-pending-transactions.ts", "setup.ts"]}>
  <Tabs.Tab>
  ```typescript 
    const pendingTransactions = await safeClient.getPendingTransactions()
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```typescript 
    import { createSafeClient } from '@safe-global/sdk-starter-kit'  

    const safeClient = await createSafeClient({
      provider,  
      signer,
      safeAddresss
    })
  ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`SafeMultisigConfirmationListResponse`

A paginated list of the pending transactions.
