# `isOwner`

Checks if a specific address is an owner of the current Safe.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["is-owner.ts", "setup.ts"]}>
  <Tabs.Tab>
    ```typescript 
      const ethereumAddress = '0x...'
      const isOwner = await safeClient.isOwner(ethereumAddress)
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript 
      import { createSafeClient } from '@safe-global/sdk-starter-kit'

      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Parameters

### `ownerAddress`

- **Type:** `string`

The ethereum address to check if it is an owner of the Safe.

## Returns

`Promise<boolean>`

A boolean value indicating if the provided address is an owner of the Safe.
