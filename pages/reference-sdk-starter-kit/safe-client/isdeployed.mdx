# `isDeployed`

Verifies the deployment status of the existing Safe.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript 
    import { safeClient } from './setup.ts'
    
    const isSafeDeployed = await safeClient.isDeployed()
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import { createSafeClient } from '@safe-global/sdk-starter-kit'
    
    export const safeClient = await createSafeClient({
      provider,
      signer,
      safeAddress
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<boolean>`

A boolean value indicating if the Safe is deployed.
