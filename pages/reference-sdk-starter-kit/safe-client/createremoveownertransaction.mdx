# `createRemoveOwnerTransaction`

Creates a transaction (to, value, data) to remove an owner to the Safe. The transaction can be included individually or in a batch within the `send` method.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    import { safeClient } from './setup.ts'

    const transaction = await safeClient.createRemoveOwnerTransaction({
      ownerAddress,
      threshold
    })

    const txResult = await safeClient.send({
      transactions: [transaction]
    })
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
    import { createSafeClient } from '@safe-global/sdk-starter-kit'
    
    export const safeClient = await createSafeClient({
      provider,
      signer,
      safeAddress
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<TransactionBase>`

Returns a `TransactionBase` transaction that you can use or include in a batch using the `send` method.

## Parameters

### `ownerAddress`

- **Type:** `string`

The Ethereum address to remove as an owner.

### `threshold` (optional)

- **Type:** `string`

The new threshold for the Safe account.
