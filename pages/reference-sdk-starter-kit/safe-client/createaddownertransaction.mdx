# `createAddOwnerTransaction`

Creates a transaction (to, value, data) to add an owner to the Safe. The transaction can be included individually or in a batch within the `send` method.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['create-add-owner-transaction.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript
    const transaction = await safeClient.createAddOwnerTransaction({
      ownerAddress,
      threshold
    })

    const txResult = await safeClient.send({ transactions: [transaction] })
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript
      import { createSafeClient } from '@safe-global/sdk-starter-kit'
          
      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<TransactionBase>`

Returns a `TransactionBase` transaction that you can use or include in a batch using the `send` method.

## Parameters

### `ownerAddress`

- **Type:** `string`

The Ethereum address to add as an owner.

### `threshold` (optional)

- **Type:** `string`

The new threshold for the Safe account. If not provided, the current threshold is used.
