# `getAddress`

This method retrieves the current Safe address.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["get-address.ts", "setup.ts"]}>
  <Tabs.Tab>
    ```typescript 
      const safeAddress = await safeClient.getAddress()
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript 
      import { createSafeClient } from '@safe-global/sdk-starter-kit'

      const safeClient = await createSafeClient({
        provider,  
        signer,
        safeAddress
      })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`Promise<string>`

The Safe address or the predicted one, in case it has not yet been deployed. You can use the `safeOptions` to instantiate the SafeClient and to produce a deterministic address based on the different options.
