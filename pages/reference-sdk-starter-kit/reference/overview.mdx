# SDK Starter Kit Reference

The [SDK starter kit](https://github.com/safe-global/safe-core-sdk/tree/main/packages/sdk-starter-kit) facilitates the interaction with the Safe protocol

## Install dependencies

To add the API Kit to your project, run:

{/* <!-- vale off --> */}

<CH.Section>
  <CH.Code style={{ boxShadow: 'none' }}>
    ```bash pnpm
    pnpm add @safe-global/sdk-starter-kit
    ```

    ```bash npm
    npm install @safe-global/sdk-starter-kit
    ```

    ```bash yarn
    yarn add @safe-global/sdk-starter-kit
    ```

  </CH.Code>
</CH.Section>
