# `getPendingSafeOperations`

Retrieves the pending Safe operations for the current Safe account.

## Usage

{/* <!-- vale off --> */}

<Tabs items={['example.ts', 'setup.ts']}>
  <Tabs.Tab>
    ```typescript 
    import { safeOperations } from '@safe-global/sdk-starter-kit'
    import { safeClient } from './setup.ts'

    const safeOperationsClient = await safeClient.extend(
      safeOperations({ bundlerUrl }, { isSponsored, paymasterUrl })
    )

    const pendingSafeOperations = await safeClientWithSafeOperation.getPendingSafeOperations()
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```typescript 
    import { createSafeClient } from '@safe-global/sdk-starter-kit'

    export const safeClient = await createSafeClient({
      provider,
      signer,
      safeAddress
    })
    ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`GetSafeOperationListResponse`

A paginated list of the pending SafeOperations for the current Safe account.
