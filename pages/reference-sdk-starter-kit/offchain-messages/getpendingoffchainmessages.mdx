# `getPendingOffChainMessages`

This method returns the list of pending off-chain messages that require confirmation by using the `confirmOffChainMessage` method.

## Usage

{/* <!-- vale off --> */}

<Tabs items={["get-pending-off-chain-messages.ts", "setup.ts"]}>
  <Tabs.Tab>
  ```typescript 
    import { offChainMessages } from '@safe-global/sdk-starter-kit'
    
    const safeMessageClient = safeClient.extend(offChainMessages())
    const pendingMessages = await safeMessageClient.getPendingOffChainMessages()
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```typescript 
    import { createSafeClient } from '@safe-global/sdk-starter-kit'

    const safeClient = await createSafeClient({
      provider,  
      signer,
      safeAddress
    })  
  ```
  </Tabs.Tab>
</Tabs>

{/* <!-- vale on --> */}

## Returns

`SafeMessageListResponse`

A paginated list of the pending off-chain messages
