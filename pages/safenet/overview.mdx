# Safenet Overview

Safe announced Safenet on December 3rd 2024. These docs describe the current state of technical vision. The current state of implementation can be found in the roadmap. 

The docs consist of the following sections:

### Introduction

Provides a brief overview of Safenet, its high-level architecture, and the flow of a Safenet transaction. Reading this part will provide an understanding of why Safenet is built.

It is recommended to start with this part in the given order.

### Core Components

Provides details of the different components of Safenet. Reading this part will provide an understanding of what Safenet is composed of.

### Concepts

Describes the different concepts and procedures of Safenet. Reading this part will provide an understanding of how Safenet works.

### Protocol

Describes the smart contracts and their role in the Safenet protocol. Reading this part will provide an understanding of the technical details of Safenet.
